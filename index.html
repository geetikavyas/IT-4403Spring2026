Site Description

This website serves as a demonstration of Milestone 1 for the IT 4403 Individual Project. It shows how JSON data can be read using JavaScript and jQuery and displayed in a clean and organized format.

This demo is provided to help students understand expectations. Students are encouraged to be creative and design their own unique layout and style rather than copying this example.
