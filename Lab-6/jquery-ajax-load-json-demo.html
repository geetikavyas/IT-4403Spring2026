jQuery AJAX loading JSON using .getJSON(), .get(), .ajax()

This file demonstrates jQuery AJAX with JSON data, where button clicks request JSON from the server and display instructor and course information without reloading the page. It is different from the previous two files because those loaded HTML content, while this one loads JSON data and then uses JavaScript/jQuery to pick specific values and show them on the page.

The server side JSON can be viewed here:

  • server-data-json.php
  • server-data-json.php?withcourse=true

The following three clickable items demonstrate three ways to trigger the click event and stay on the same page

| Click me to load instructor JSON data | Click me to load instructor with course JSON data

Summary

.getJSON() → simplest simplest when the server returns JSON; automatically parses JSON i.e.when you know the response is JSON

.get() → simple and short way to send a GET request for data.

.ajax() → most flexible and most powerful, allows full control over (parameters, headers, data types, and options).
