Raw JavaScript AJAX Demo: Loading Server Content Without Page Reload

This demo shows how raw JavaScript AJAX (XMLHttpRequest) can request static and dynamic content from the server and update specific sections of a webpage without reloading the entire page. It demonstrates loading a static HTML response and a dynamic PHP response with parameters. This code uses raw JavaScript (XMLHttpRequest), showing how AJAX works at a lower level without libraries.
