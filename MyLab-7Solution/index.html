Lab 7: AJAX & REST Examples

This page demonstrates different ways of using AJAX, JSON, and APIs in web applications. Each example shows a different technique such as JSONP, CORS, and API integration.

1. Bing Search Suggestions

This example uses the Bing Suggest API to fetch search suggestions dynamically. It demonstrates how JSONP is used to handle cross-domain requests and display results as the user searches.

Open Bing Suggest Demo

2. OpenLibrary Book Search

This example uses the OpenLibrary API to search for books by title. It demonstrates how to retrieve JSON data and display structured results including title, author, and cover image.

Open Book Search Demo

3. Product List (CORS Example)

This example uses a public API to fetch a list of products. It demonstrates how CORS-enabled APIs allow direct AJAX requests without JSONP.

Open Product List Demo

4. ZIP Code Lookup (JSONP)

This example looks up city and state information using a ZIP code. It demonstrates JSONP for cross-domain API requests and dynamically displays location data.

Open ZIP Lookup Demo

Understand how to use AJAX, JSON, JSONP, and REST APIs to build dynamic and interactive web applications.
