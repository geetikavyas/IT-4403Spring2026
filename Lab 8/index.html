Lab 8: CSS Flexbox and Mustache Template

This lab demonstrates how to use CSS Flexbox for layout design and Mustache templates for displaying JSON data. Students will build dynamic web pages that display book data and support multiple layout views.

1. Flexbox Layout Practice

This part introduces the basics of CSS Flexbox. Students create a simple layout using flex container and items. It shows how changing item width can switch between flow list and grid layout.

Open Flexbox Demo

2. Google Books API (Grid Layout)

This example uses the Google Books API to search for books and display results. Books are shown in a grid layout using CSS Flexbox.

Open Book Grid Demo

3. Layout Switching (Flexbox)

This example adds buttons to change layouts dynamically. Users can switch between flow list, grid view, and different item arrangements.

Open Layout Switch Demo

4. Mustache Template (Book Grid)

This example improves the previous API display by using Mustache templates. It separates data from UI and makes the code cleaner and easier to maintain.

Open Mustache Grid Demo

5. Layout Switching with Mustache

This example combines layout switching with Mustache templates. It supports multiple views while keeping the UI rendering clean and structured.

Open Mustache Layout Demo

6. Single Book View (Template Switching)

This example displays a single book using two different templates. Users can switch between views using buttons to see different UI designs.

Open Single Book View Demo

Learning Objective

Understand how to use CSS Flexbox for responsive layouts and Mustache templates to display JSON data. Learn how to build interactive web applications with dynamic layout switching.
